Doctrine in Apigility Documentation
===================================

Welcome to the Doctrine in Apigility Book.  This documentation will guide you from the 
`zfcampus/zf-apigility-skeleton <https://github.com/zfcampus/zf-apigility-skeleton>`_ to 
a fully `Richardson Maturity Model Level 3 <https://martinfowler.com/articles/richardsonMaturityModel.html>`_
`HATEOAS <https://en.wikipedia.org/wiki/HATEOAS>`_ API built with the 
`Doctrine Object Relational Mapper <http://docs.doctrine-project.org/projects/doctrine-orm/en/latest/>`_.
